# Daily Diary: 2026-04-03

## Topic: Streaming Responses and Async LLM Calls

Implemented streaming responses to reduce perceived latency in chat applications.
Used `asyncio` and `AsyncOpenAI` client to run multiple LLM requests concurrently. This is critical for scale, preventing sequential blocking when processing batches of customer reviews.

**Code Snippet of the Day:**
```python
import asyncio
from openai import AsyncOpenAI

async def fetch_summary(client: AsyncOpenAI, text: str):
    response = await client.chat.completions.create(
        model="gpt-4o-mini",
        messages=[{"role": "user", "content": f"Summarize: {text}"}]
    )
    return response.choices[0].message.content
```

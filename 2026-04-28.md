# Daily Diary: 2026-04-28

## Topic: Query Rewriting and Expansion

Implemented query rewrite techniques.
User queries are often short or poorly worded, which results in bad vector retrieval. I added a pre-retrieval step where an LLM expands a single query into 3 alternative formulations.
Retrieving documents for all 3 and taking the union drastically improved recall of relevant information.

# Daily Diary: 2026-04-24

## Topic: Metadata Filtering in Vector Databases

Learned how to use metadata filtering in vector queries.
Instead of doing a broad semantic search, I restricted the search space using metadata filters (e.g., `{'author': 'Jane Doe'}` or `{'date': {'$gt': '2026-01-01'}}`).
This improves search speed and context precision, reducing noise in the final LLM prompt.

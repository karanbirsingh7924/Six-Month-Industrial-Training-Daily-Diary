# Daily Diary: 2026-05-05

## Topic: Caching Embeddings to Prevent Redundant API Calls

Embedding APIs cost money.
Implemented a caching layer where the MD5 hash of text chunks is checked against a Redis/Local cache before querying the embedding API.
If the hash matches, the existing embedding is retrieved from cache. This reduced api latencies and costs for bulk document updates by 40%.

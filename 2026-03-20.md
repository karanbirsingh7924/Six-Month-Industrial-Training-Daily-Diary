## Topic: Building a Robust ETL Pipeline with Pandas and Pydantic

Combined pandas preprocessing with Pydantic validation.
Created a data loader pipeline that reads a CSV, converts rows to dictionaries, validates each row using a Pydantic model, catches validation errors, and logs corrupt rows to a separate error file.

**Architecture:**
- Read CSV in chunks.
- Validate rows via Pydantic.
- Filter valid rows into a clean DataFrame.
- Write invalid rows to `quarantine_log.json`.

# Daily Diary: 2026-03-18

## Topic: Introduction to Pydantic v2 for Data Validation

Started exploring Pydantic v2. Pydantic is an excellent tool for data parsing and validation, which is crucial before feeding data into ML models or LLMs.
Created my first `BaseModel` to validate configuration settings and row-level structures for tabular data. Noticed the performance improvements in Pydantic v2 (written in Rust).

**Code Snippet of the Day:**
```python
from pydantic import BaseModel, Field

class CustomerRecord(BaseModel):
    customer_id: int
    age: int = Field(..., ge=18, le=100)
    income: float = Field(..., gt=0)
    churned: bool
```

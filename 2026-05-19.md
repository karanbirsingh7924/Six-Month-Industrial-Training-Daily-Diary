# Daily Diary: 2026-05-19

## Topic: Introduction to LangGraph for Agent Workflows

Started using LangGraph to construct stateful, multi-actor agent systems.
LangGraph Models workflows as graphs where nodes are actions (Python functions) and edges are transitions (conditional logic).
Configured a state object to store context and chat history, passing it from node to node. This makes managing complex flows much cleaner than raw ReAct loops.

# Daily Diary: 2026-04-27

## Topic: Evaluating RAG Quality: Faithfulness and Relevancy

RAG systems suffer from hallucinations.
Implemented metrics inspired by RAGAS to evaluate our pipeline:
- Faithfulness: Is the answer derived *only* from the context?
- Answer Relevancy: Does the answer address the user query?
- Context Precision: Were all retrieved chunks actually relevant?

Developed Pydantic models to parse the evaluation results.

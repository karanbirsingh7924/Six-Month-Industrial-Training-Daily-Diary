# Daily Diary: 2026-04-18

## Topic: Dimensionality Reduction on Embeddings

Connected embeddings with classical data science.
Generated embeddings for 200 documents, and then used t-SNE (t-Distributed Stochastic Neighbor Embedding) and PCA (Principal Component Analysis) from `scikit-learn` to project these high-dimensional (1536-dim) vectors onto 2D space.
Plotted them using matplotlib. Clear clusters emerged corresponding to different document topics!

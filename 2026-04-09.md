# Daily Diary: 2026-04-09

## Topic: LLM Self-Correction Loop with Pydantic Validators

Implemented validation retry logic.
If a Pydantic validator fails (e.g., if extracted email is invalid or a numeric value is out of range), the validation error is caught, and the error message is fed back to the LLM. The LLM then self-corrects and tries to output a corrected structure.
`instructor` supports this natively via `max_retries`!

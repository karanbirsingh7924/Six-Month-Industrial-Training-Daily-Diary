## Topic: Integrating Scikit-learn Predictions with LLM Context

Built a hybrid pipeline today.
Input data is passed through our Scikit-Learn churn model. If the churn prediction is high (>0.8), the user record and prediction details are formatted into a prompt context.
The LLM is then invoked to generate a customized retention email containing key points of interest. This combines predictive ML with generative AI.

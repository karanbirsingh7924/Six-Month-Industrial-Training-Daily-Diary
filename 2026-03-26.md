# Daily Diary: 2026-03-26

## Topic: Baseline Modeling with Logistic Regression

Trained my first baseline classification model using Scikit-Learn's `LogisticRegression`.
Calculated baseline metrics: Accuracy, Precision, Recall, F1-Score, and ROC-AUC.
Realized that accuracy is highly misleading due to class imbalance (90% non-churn, 10% churn). Precision-Recall AUC is a much better metric here.

## Topic: Nested Pydantic Models for Complex LLM Extraction

Used nested Pydantic models with the `instructor` library to parse complex entities from unstructured text.
For example, parsing a medical report into an `Invoice` containing a list of `LineItem` objects, each containing quantity, description, unit price, and taxonomy codes.
This showcases the expressive power of combining Pydantic with LLMs.

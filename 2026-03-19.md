# Daily Diary: 2026-03-19

## Topic: Pydantic Custom Validators and TypeAdapter

Learned about `@field_validator` and `@model_validator` in Pydantic v2.
I implemented custom validation rules to check if target columns match specific criteria (e.g., ensuring a registration date is in the past).
Also used `TypeAdapter` to validate lists of dicts directly parsed from external JSON files.

**Key Learning:**
Model validators are useful when fields depend on each other (e.g., if `has_premium_subscription` is True, `subscription_cost` must be greater than zero).

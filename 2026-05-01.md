# Daily Diary: 2026-05-01

## Topic: Reranking Retrieved Chunks using Cohere

Retrieving 10 chunks can overwhelm the context window and dilute key facts.
Implemented a reranking step using Cohere's rerank API. I retrieve 20 documents using fast vector search, and then pass them through the reranker to select the top 5 most highly aligned chunks.
This preserves computational cost while enhancing answer quality.

# Daily Diary: 2026-05-18

## Topic: Building a ReAct (Reasoning and Action) Loop from Scratch

Built a ReAct loop manually to understand the core mechanism.
The loop goes: User query -> LLM thinks (Thought) -> LLM chooses action (Action) -> Execute action and get result (Observation) -> Feed back to LLM -> LLM generates final answer (Answer).
This loop runs in a `while` loop until a final answer is determined.

**Key Learning:**
ReAct prompts need strict parsing logic to avoid infinite loops when the LLM outputs unexpected formats.

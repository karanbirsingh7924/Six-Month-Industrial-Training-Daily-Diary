# Daily Diary: 2026-04-07

## Topic: Introduction to Instructor Library

Discovered the `instructor` library by Jason Liu, which simplifies structured LLM outputs using Pydantic.
It patches the OpenAI client so that it accepts a Pydantic model in the `response_model` argument. Under the hood, it handles tool calling, extracts the arguments, and returns a fully parsed Pydantic object!

**Code Snippet of the Day:**
```python
import instructor
from openai import OpenAI
from pydantic import BaseModel

class UserDetails(BaseModel):
    name: str
    age: int

client = instructor.patch(OpenAI())
user: UserDetails = client.chat.completions.create(
    model="gpt-4o-mini",
    response_model=UserDetails,
    messages=[{"role": "user", "content": "Jason is 25 years old"}]
)
```

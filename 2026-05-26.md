# Daily Diary: 2026-05-26

## Topic: Asynchronous Endpoints and Background Tasks

Document ingestion and vector indexing take time. Running them synchronously blocks the API server.
Implemented FastAPI's `BackgroundTasks` to start ingestion processes asynchronously, immediately returning a `202 Accepted` status along with a job ID so clients can poll for status.

**Code Snippet of the Day:**
```python
from fastapi import FastAPI, BackgroundTasks

app = FastAPI()

@app.post("/ingest")
async def start_ingestion(background_tasks: BackgroundTasks, doc_id: str):
    background_tasks.add_task(process_document, doc_id)
    return {"status": "processing", "job_id": doc_id}
```

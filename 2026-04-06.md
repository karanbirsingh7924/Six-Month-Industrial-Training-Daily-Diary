# Daily Diary: 2026-04-06

## Topic: JSON Mode vs. Function Calling

Compared OpenAI's Native JSON Mode and Function Calling (Tool Calling).
JSON Mode ensures output is syntactically valid JSON, but doesn't guarantee compliance with a specific schema.
Tool Calling, however, allows us to supply a JSON schema representing parameters. The model matches the schema structure, which is much more reliable for programmatic data extraction.
